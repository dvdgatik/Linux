Los archivos .tar.gz son archivos comprimidos en Linux


para descomprimirlo se usa

tar -xvf + nombre del archivo.tar.gz

Al descomprimirlo aparecera un archivo de color verde llamado configure esto queire decir que se puede ejecutar 
con ./configure
despues se utiliza make para preparar la instalacion
despues sudo make install



para saber la verison de un programa se utiliza nombredelproggrama +  -v
para saber donde esta un programa se utiliza whereis + nombredelprograma


Instalacion de paquetes por extension .sh --------------

parta ejecutar un script.sh se debe hacer asi:

bash nombre del script.sh
debe tener permisos de ejecucion x
./ se ejecuta
